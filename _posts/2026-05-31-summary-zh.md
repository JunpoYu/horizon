## [交互式文章详解 1992 年《科曼奇》游戏的地形渲染算法](https://s-macke.github.io/VoxelSpace/) ⭐️ 7.0/10

S. Macke 发布了一篇交互式技术文章，解释并模拟了 1992 年游戏《科曼奇》中使用的地形渲染算法，并澄清该算法基于高度图而非真正的体素。文章包含一个网页演示，并链接了该算法在现代的 C++和 AGS 引擎等环境中的移植版本。 这次深度解析很重要，因为它保存并阐明了一项具有历史意义的渲染技术，该技术在当时是突破性的，能在 1990 年代的 CPU 上实现细致的地形。它为理解 3D 图形的发展提供了宝贵的教育材料，并以高效的软件渲染概念启发现代开发者。 该算法常被误称为“体素”技术，但实际上它通过从后向前绘制像素垂直列来渲染高度图，这种技术被称为“光线投射”。原始引擎完全用 x86 汇编语言编写，以在当时有限的硬件上实现最大性能。

hackernews · davikr · May 30, 14:25 · [社区讨论](https://news.ycombinator.com/item?id=48336564)

**背景**: 1992 年，3D 游戏完全依赖 CPU 渲染，没有 GPU 加速。NovaLogic 的《科曼奇》是一款以其细致地形而闻名的飞行模拟器，它通过其专有的“Voxel Space”引擎实现了这一效果。高度图是一个二维数组，其中每个值代表该点的高度，常用于地形渲染；而真正的体素则是通过在三个轴上均等地划分三维空间来表示体积。

Anthropic 发布了全面的文档，详细说明了在不同产品（包括 Claude.ai、Claude Code 和 Claude Cowork）中用于约束其 Claude AI 模型的具体沙箱技术。该文档解释了如何使用 gVisor、Seatbelt、Bubblewrap 和完整虚拟机等工具来创建安全边界。 这具有重要意义，因为关于沙箱的详细、透明的文档非常罕见，并且它解决了随着 AI 智能体获得更多与系统和数据交互能力而产生的关键安全问题。它为保护 AI 应用程序安全提供了一个实用、可信的蓝图，这对于智能体 AI 的广泛采用和安全部署至关重要。 具体实现因产品而异：Claude.ai 使用 Google 的 gVisor 沙箱；本地运行的 Claude Code 使用 macOS 的 Seatbelt 和 Linux 的 Bubblewrap；而 Claude Cowork 则在完整虚拟机中运行，使用 Apple 的 Virtualization 框架或 Windows HCS。该文档还坦率地讨论了过去的安全疏忽，例如通过 `api.anthropic.com/v1/files` 的文件泄露途径。

## [通过 Pyodide 和 Service Worker 在浏览器中运行 Python ASGI 应用，实现完整 JavaScript 执行。](https://simonwillison.net/2026/May/30/pyodide-asgi-browser/#atom-everything) ⭐️ 7.0/10

开发者 Simon Willison 成功展示了一个研究项目，该项目使用 Pyodide 和 Service Worker 在浏览器中完全运行 Python ASGI 应用，克服了之前基于 Web Worker 方法无法执行 JavaScript 的限制。这一成果是通过让 AI（Claude Opus 4.8）来探索方法实现的，并产生了包括一个基础 ASGI 应用和 Datasette 1.0a31 在内的功能演示。 这一突破意义重大，因为它使得完整的 Python 全栈 Web 应用（包括那些带有交互式 JavaScript 组件的应用）能够在客户端运行，而无需服务器后端。这代表了 Datasette Lite 等浏览器内 Python 工具的重大进步，使得复杂的插件和动态功能能够按预期工作，从而扩展了复杂、无服务器 Web 应用的潜力。 新的 Service Worker 方法会拦截网络请求，并提供由 Pyodide 托管的 Python ASGI 应用生成的响应，从而允许嵌入的 `<script>` 标签正常执行。该项目目前是一个研究原型，作者计划在进一步理解后，将这项技术集成到 Datasette Lite 的生产版本中。

**背景**: ASGI（异步服务器网关接口）是异步 Python Web 应用与服务器之间的标准接口，被 FastAPI 和 Django Channels 等框架使用。Pyodide 是将 CPython 移植到 WebAssembly 的项目，允许 Python 及其科学计算栈（如 NumPy）直接在浏览器中运行。Service Worker 是一种 JavaScript 脚本，充当网页的可编程网络代理，支持离线缓存和拦截 fetch 请求等功能。
